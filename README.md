
# South Africa Move Planner

## Project Overview

This project helps individuals planning an international relocation estimate their savings goals and relocation timeline.

## Features

- Calculates remaining savings needed
- Estimates months required to reach a target amount
- Assists with relocation budgeting
- Supports financial planning

## Example

Target Savings: R100,000

Current Savings: R27,000

Monthly Savings: R10,000

Remaining Amount: R73,000

Estimated Time: 7.3 Months
